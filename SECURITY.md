# Security Policy

## Supported Versions

We currently support and patch only the latest stable release of WiGuard:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in WiGuard, please do not open a public issue. Instead, report it via email to the maintainer:

- Maintainer: **Dino20004**
- Email: **dino2004@gmail.com**

Please include the following details in your report:
- Type of issue (e.g., buffer overflow, SQL injection, privilege escalation).
- Detailed steps to reproduce the vulnerability.
- Potential impact of the vulnerability.

We will acknowledge receipt of your report within 48 hours and strive to provide a resolution within 14 days.
